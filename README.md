Описание программы
Запуск - инфиализация класса Contacts5Application
После запуска программа предлагает нажать одну из ЦИФр от 1 до 5
Функционал:
- Показать все контакты (производится считывание контактов из файла и вывод на экран)
- Добавить контакт (необходимо ввести строку с разделителями ; которая будет засчитана за данные нового контакта)
- Удалить контакт по email (необходимо ввести email, будет сделан поиск контактов с таким email и их удаление)
- Сохранить контакты в файл (идёт сохранение контактов из памяти в файл)
- Выйти 
Все некорректные вводы обрабатываются и сообщаются пользователю.
Место хранения файла с контактами для загрузки, а также место сохранения данных в файл храниться в свойствах приложения application.properties
Есть возможность отключить первоначальное считвание контактов из файла. Нужно в настроечном файле application.properties изменить значение defaultContacts на любое другое.
